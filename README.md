# PL-300-Microsoft-Power-BI-Data-Analyst-Study-Resources
This repository is used to house various resources that may help you prepare for **Exam PL-300: Microsoft Power BI Data Analyst**. These resources may be referenced in the course videos, but it **IS NOT NECESSARY** to follow along with all these files during the lessons. The files are provided just in case you want to try a few things out for yourself. 

Here is a description of each of the directories:

**Data** contains many of the raw data files that are used to populate the Power BI pbix files. For the pbix files to see these files, you will need to put the folder at C:\PL-300\ on your machine. If you use another location, you will need to change the data sources in each pbix file to wherever you put the data folder in your environment. For more details, see the lesson titled Change data source settings, including credentials and privacy levels. This is a testable exam skill.

Note: Some lessons use an Adventure Works SQL Server Database, plus a few other sources. There is no need to get this database and go through the work of installing it for the handful of lessons that use it. It is recommended to just watch any lessons that use data that IS NOT in the Data directory as no guidance or instructions are given on how to do this. It is not worth the effort. 

**pbix** contains many of the pbix files that are used in the lessons

**Template** contains template files that are used in a few of the lessons
